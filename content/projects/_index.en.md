This section contains documentation and source code for projects developed to address challenges in system administration, infrastructure optimization, and workflow automation.

All projects are Open Source, documented, and focused on practical utility.

## Areas of Work

### 📦 Infrastructure & Self-Hosting

Solutions for independent service hosting. Server configuration management, fault tolerance, and security hardening.

- _Technologies_: NixOS, Ansible, Docker Compose, OpenWRT, WireGuard.
- _Examples_: Home server configuration, service status pages, router configurations.

### 🚀 Containerization & ARM Optimization

Development and optimization of Docker images for ARM64 architecture. Adapting server software for single-board computers (Orange Pi, Raspberry Pi) with limited resources.

- _Technologies_: Docker Buildx, Multi-stage builds, QEMU.
- _Examples_: ARM-based game servers (GMod, Minecraft), lightweight databases, proxy servers.

### 🤖 Automation & Bots

Development of modular bots for messengers and tools for routine automation. Integration with local LLMs and external APIs.

- _Technologies_: Python, PHP, Telegram Bot API, Matrix Protocol, Ollama.
- _Examples_: AI chat assistants, monitoring scripts, notification utilities.

### 🌐 Networking & Hardware

Experiments with wireless mesh networks, network equipment configuration, and hardware integration.

- _Technologies_: LoRa, Meshtastic, ESP32, OpenWRT, VLAN.
- _Examples_: Mesh node configuration, access point setup, IoT sensors.
